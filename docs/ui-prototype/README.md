# UI原型说明文档

## 目录结构
- frontend/     # 前台用户界面原型
  - index.html  # 首页
  - editor.html # 视频编辑页面
  - template.html # 模板列表页面
  - profile.html # 用户中心页面
  
- admin/        # 后台管理界面原型
  - login.html  # 登录页面
  - dashboard.html # 控制台首页
  - users.html    # 用户管理页面
  - templates.html # 模板管理页面

## 使用说明
1. 所有页面采用纯HTML+TailwindCSS开发
2. 使用Font Awesome图标库
3. 采用响应式设计，适配移动端
